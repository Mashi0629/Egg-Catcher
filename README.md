#  Egg Catcher

A fun egg-catching arcade game built with Python and Pygame. Move your basket to catch falling eggs, earn points, and survive all 5 levels!

---

##  Gameplay

- Eggs fall from the top of the screen — catch them with your basket!
- Golden eggs are rare but worth 5 points each
- Miss an egg and you lose a life ❤️
- Survive all 5 levels to win

---

##  Controls

| Key | Action |
|-----|--------|
| `←` Left Arrow | Move basket left |
| `→` Right Arrow | Move basket right |
| `Space` / `Enter` | Start / Continue |
| `Esc` | Quit |

---

##  Levels

| Level | Difficulty | Eggs to Clear |
|-------|-----------|---------------|
| 1 | Easy | 10 |
| 2 | Medium | 20 |
| 3 | Hard | 30 |
| 4 | Expert | 40 |
| 5 | Insane | 50 |

Speed and spawn rate increase with each level. Good luck on Level 5! 

---

##  Getting Started

### Prerequisites

- Python 3.7+
- pip

### Installation

```bash
# Clone the repository
git clone https://github.com/mashi0629/egg-catcher.git
cd egg-catcher

# Install dependencies
pip install pygame
```

### Run the Game

```bash

python egg_catcher.py
```
